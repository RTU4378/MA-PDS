# MA-PDS

This project proposes model-adaptive Poisson-disc sampling (MA-PDS), a sampling-mask generation framework for accelerated Cartesian MRI. Instead of optimizing arbitrary point-wise k-space masks, MA-PDS directly constructs binary phase-encoding line masks that preserve the auto-calibration signal region, maintain fully sampled readout, enforce an exact sampling budget, and improve mask-level spacing regularity. The method estimates a phase-encoding sensitivity map from a fixed pretrained reconstruction model and converts this model-specific sensitivity prior into a variable-interval Poisson-disc sampling process. Experiments on brain and knee MRI datasets show that MA-PDS improves retrospective reconstruction performance under spacing-constrained Cartesian undersampling, especially at high acceleration factors. This framework provides a practical route toward reconstruction-aware PE-line sampling, while prospective scanner validation remains necessary before clinical deployment.
